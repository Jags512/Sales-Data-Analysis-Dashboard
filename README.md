# Sales Data Analysis & Dashboard

## Overview
This project analyzes sales data to extract insights such as revenue trends, top products, and regional performance.

## Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI / Excel

## Key Features
- Data Cleaning & Preprocessing
- KPI Extraction (Revenue, Top Product)
- Data Visualization
- Dashboard Creation

## Results
- Identified top-performing products
- Analyzed region-wise sales performance
- Generated actionable insights for business decisions

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run script:
   python scripts/analysis.py
